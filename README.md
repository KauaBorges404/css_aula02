# 📚 Atividades de Desenvolvimento Web

Este repositório contém um conjunto de projetos práticos desenvolvidos como parte das atividades acadêmicas do curso. O objetivo é aplicar conceitos fundamentais de estruturação de páginas, estilização avançada, tipografia e legibilidade no ecossistema Web usando HTML5 e CSS3.

---

## 💻 Atividades Incluídas

### 1. Portal de Notícias Simples (Blog News)
Uma página web básica que simula um feed de notícias tecnológico focado em linguagens de programação.
* **Tecnologias:** HTML5 e CSS3 (Estilização interna).
* **Conceitos Aplicados:**
  * Estruturação de layouts usando blocos (`<div>`).
  * Estilização com seletores de ID (`#logo`) e Classe (`.noticia`).
  * Uso de fundos com gradiente radial (`radial-gradient`) aplicado a elementos de cabeçalho.
  * Alinhamento centralizado e espaçamento de blocos usando `margin: auto` e `padding`.

### 2. Portal de Tecnologia Avançado (Portal)
Uma evolução da atividade anterior, focada em tendências reais do mercado (IA Multimodal, Computação Quântica, BCIs) com um design de interface mais robusto, moderno e focado na legibilidade de textos longos.
* **Tecnologias:** HTML5 e CSS3.
* **Conceitos Aplicados:**
  * Estilização global do plano de fundo da página (`body`) com gradiente radial adaptável.
  * Criação de contêineres no formato de cartões (*cards*) usando bordas espessas e cantos muito arredondados (`border-radius: 50px`).
  * Customização tipográfica fina, alterando o tamanho da fonte (`font-size: 30px`) e definindo famílias de fontes sem serifa (`font-family`) para melhor leitura.

### 3. Página de Receitas Estruturada (Brownies Perfeitos)
Uma página focada na exibição organizada e semântica de dados, utilizando estruturas de listas para separar ingredientes e o modo de preparo de uma receita culinária.
* **Tecnologias:** HTML5 e CSS3.
* **Conceitos Aplicados:**
  * Uso de tags semânticas de listagem: Listas Não Ordenadas (`<ul>`) para a exibição dos ingredientes e Listas Ordenadas (`<ol>`) para indicar a sequência do passo a passo.
  * Aplicação de gradiente linear (`linear-gradient`) com definição de ângulo de direção (180 graus) ocupando todo o plano de fundo.
  * Controle de fluxo de espaçamento vertical usando margens inferiores (`margin-bottom: 15px`) para evitar colisão entre blocos de conteúdo.
  * Modelagem de caixas de conteúdo (*boxes*) com bordas pretas sólidas e arredondamento suavizado (`border-radius: 20px`).
